# Roadmap

## Near-term

- Enable database-backed persistence as the default runtime
- Add broader API and UI test coverage
- Improve evidence handling and richer run details
- Strengthen data modeling around review and lessons workflows

## Mid-term

- Add authentication and multi-user support
- Add role-aware access control
- Add import and export workflows for benchmark data
- Add richer dashboard analytics and trend views

## Longer-term

- Add approval workflows for higher-risk actions
- Add attachments and richer benchmark evidence support
- Add deployment guidance for production environments
- Expand the app into a more complete benchmark operations platform
